Sirpinski
What is it?
This project displays three different ways to create a Sirpinski shape: equilateral triangle, right triangle, and trapezoid. 
How to Use It
Click the RESTART button any time you want a shape.
Click the SETUP and then the PLAY button pair for whichever shape you prefer.
Be sure to click the RESTART button again before starting another shape.
