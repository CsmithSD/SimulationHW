Ant- Trails

What is it?

In this simple program, each ant takes a step forward, toggles the color of the square it lands on, and turns left or right depending on the square's original color. Complex patterns emerge from this simple rule. Try experimenting with the initial positions of the ants to see different patterns result.
How to Use It

Click the setup button to place the two ants. Click the go button to start the simulation.
Things to Notice

Watch how the ants begin to form separate patterns. How does it change when they overlap? Do the patterns stay there once they've been placed?
Explorations
  • Try changing the number of ants. How does the number of ants change the patterns?
  • Try changing how the ants move - which way they turn and how far.
