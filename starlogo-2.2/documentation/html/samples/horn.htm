Horn

What is it?

With only a few StarLogo commands, it's simple to make graphics. This particular program creates a horn object which gets larger and moves across the screen.
How to Use It
Simply press horn or horn1 in the interface window and watch the horns form.
Things to Notice

This program uses simple rules to create a simple picture, can you make more complex graphics?
