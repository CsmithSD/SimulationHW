Turtle, Observer, Patch

number1 mod number2

Parameters:
number1 Number
number2 Number

Description:
Modulo function. number1 mod number2 is equal to the remainder when number1 is divided by number2. The answer to mod is always positive.

Examples:
6 mod 2 returns 0.

10 mod 3 returns 1.

14 mod 5 returns 4.

Related Commands:
div
